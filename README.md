# Ora on Pi Network 
## **Features**

- Authentication with Pi Browser
- Donate, tip with Pi Account
- CRUD Post, Comment, Search
- Realtime Chat
- Notifications
- Follow authors, topics
- Profile
- more...

## Install

1. Clone Project
2. Install Packages (in folder server, client and socket)
3. Setup environment variables 
4. Run Ora
