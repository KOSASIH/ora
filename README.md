# Ora on Pi
## **Features**

- Authentication with Pi Browser
- Donate, tip with Pi Account
- Realtime Chat
- Notifications
- CRUD Post, Comment, Search
- Follow and unfollow author, topics
- Profile
- more...

## **Install **

1. Clone Project
2. Install Packages (each folder server, client and socket)
3. Setup environment variables 
4. Run Ora
